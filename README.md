# HACHIMON 八門 — Os Oito Portões

> O trabalho duro supera o talento. 「努力は天才に勝る。」

Planilha de treino híbrido inspirada no Maito Gai (Naruto). PWA client-side puro — sem backend, sem dependências, dados no `localStorage` do aparelho. Abre já no treino do dia; toque risca o exercício feito; um botão registra o treino (abre portões) e outro marca o cardio do dia (conta separado).

## O programa

Upper/Lower 2× por semana (Ter UA · Qui LA · Sex UB · Sáb LB), desenho híbrido:

- **Força** — lift principal pesado (4–6 reps) abrindo cada sessão, com rampa de aquecimento
- **Estética** — volume direcionado a ombro 3D, costas em V e braços (deltoide posterior 2×/semana)
- **Capacidade de trabalho** — finisher curto por sessão, periodizado por via energética (alático → glicolítico → EMOM misto → farmer carry), com progressão própria
- Sábado sem carga lombar de propósito (sexta+sábado são consecutivos)
- Double progression + trava de segurança (-10%) + deload programado/antecipável

## Os Oito Portões

Cada sessão pesada registrada acumula chakra. Os portões abrem por consistência:
Kaimon (1) → Kyūmon (8) → Seimon (16) → Shōmon (32) → Tomon (56) → Keimon (88) → Kyōmon (128) → **Shimon (176)**.
